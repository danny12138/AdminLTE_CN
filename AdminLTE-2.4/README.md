**AdminLTE** -- 响应式管理后台框架。 基于 **[Bootstrap 3](https://github.com/twbs/bootstrap)** 框架。 高度可定制、易于使用，适用于从移动设备到大型桌面不同屏幕分辨率。 现在看看效果。
